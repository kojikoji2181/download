# SonicStream - Music Download Site

## 概要
モダンでプレミアムなデザインの音楽ダウンロードサイトです。
音楽の再生とダウンロードが可能です。

## 使い方

1. **サイトを開く**
   `index.html` をブラウザ（Chrome, Edgeなど）にドラッグ＆ドロップするか、ダブルクリックして開いてください。

2. **音楽ファイルの追加**
   現在、音楽ファイルはデモ用に設定されていますが、実際のファイルは含まれていません。
   ご自身の音楽ファイルを使用するには以下の手順を行ってください：

   - `assets/music/` フォルダにMP3ファイルを配置してください（例: `track1.mp3`）。
   - `script.js` ファイルを開き、`tracks` 配列の `source` パスを配置したファイル名に合わせて変更してください。
   
   ```javascript
   const tracks = [
       {
           id: 1,
           title: "曲のタイトル",
           artist: "アーティスト名",
           cover: "assets/images/cover1.png",
           source: "assets/music/your-song.mp3", // ここを変更
           duration: "3:42"
       },
       // ...
   ];
   ```

3. **画像の変更**
   アルバムアートワークは `assets/images/` フォルダ内の画像を変更することでカスタマイズできます。

## 機能
- **再生/一時停止**: 中央のプレイヤーまたはリストから操作可能
- **シークバー**: 進行状況の変更
- **ダウンロード**: トラックリストの右側、またはプレイヤーのダウンロードボタンからファイルを保存

Enjoy your music!
